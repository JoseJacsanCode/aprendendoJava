# Aprendendo Java: Exercícios Práticos

Este repositório contém uma coleção de exercícios de programação em Java, desenvolvidos como parte de um estudo sobre os fundamentos da linguagem. O projeto está organizado em pacotes que separam os tópicos de estudo, abrangendo desde a estrutura sequencial até conceitos de programação orientada a objetos.

## Estrutura do Projeto

O espaço de trabalho está organizado da seguinte forma:

- `src`: Contém todo o código-fonte Java.
  - `estruturaSequencial`: Programas que executam instruções em ordem linear.
  - `estruturaCondicional`: Programas que utilizam estruturas de decisão (`if-else`).
  - `estruturasRepetitivas`: Programas que usam laços (`while`) para executar blocos de código repetidamente.
  - `poo`: Programas que aplicam conceitos de Programação Orientada a Objetos.

## Conteúdo dos Pacotes

### 1. Estrutura Sequencial (`estruturaSequencial`)

Programas que demonstram a leitura de dados, processamento e exibição de resultados de forma direta.

#### Exercícios:
*   **`FormasGeometricas.java`**: Lê três valores e calcula as áreas de um triângulo retângulo, círculo, trapézio, quadrado e retângulo.
*   **`SalarioFuncionario.java`**: Calcula o salário de um funcionário com base nas horas trabalhadas e no valor por hora.

### 2. Estrutura Condicional (`estruturaCondicional`)

Programas que utilizam `if-else` para tomar decisões e alterar o fluxo de execução com base em condições.

#### Exercícios:
*   **`AtribuicaoCumulativa.java`**: Calcula o valor de uma conta de telefone com base nos minutos consumidos.
*   **`IntervaloEntreNumeros.java`**: Verifica em qual de uma série de intervalos um número se encontra.
*   **`PlanoCartesiano.java`**: Determina o quadrante de um ponto (x, y) no plano cartesiano.

### 3. Estruturas Repetitivas (`estruturasRepetitivas`)

Programas que demonstram o uso de laços de repetição para resolver problemas.

#### Exercícios:
*   **`ProblemaExemploWhile.java`**: Lê números inteiros até que um zero seja digitado e, ao final, exibe a soma de todos.
*   **`ValidandoSenha.java`**: Simula um login, solicitando uma senha repetidamente até que o valor correto (2002) seja inserido.

### 4. Programação Orientada a Objetos (`poo`)

Aplicações que utilizam classes e objetos para modelar entidades do mundo real.

#### Exercícios:
*   **`Aluno.java` / `entidadesExercicios/Student.java`**: Lê o nome e as três notas de um estudante, calcula sua nota final e informa se foi aprovado ou reprovado.

## Como Executar

1.  **Pré-requisitos**:
    *   JDK (Java Development Kit) instalado.
    *   Um editor de código ou IDE, como o Visual Studio Code com o *Extension Pack for Java*.

2.  **Execução**:
    *   Abra um dos arquivos `.java` no VS Code.
    *   Clique no botão "Run" (▶) que aparece no canto superior direito do editor, ou clique com o botão direito no corpo do código e selecione "Run Java".
    *   A entrada de dados (quando necessária) deve ser feita no terminal integrado do VS Code.
